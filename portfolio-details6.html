Our team built this project using a combination of front-end and back-end technologies, including Svelte, Tailwind, NodeJS, MongoDB.

The website's front-end is powered by Svelte, Tailwind, and JavaScript, which we used to create a sleek and modern user interface. We also incorporated responsive design principles to ensure the website looks great on all devices, from desktop computers to smartphones.

On the back-end, we utilized NodeJS, ExpressJS, MongoDB to customize the website's look and feel and create dynamic content. We also integrated third-party apps and APIs, including social media plugins and payment gateways, to enhance the website's functionality and user experience.
